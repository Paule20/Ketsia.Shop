# Checklist dossier de conception

## Elements obligatoires

- Reformulation de la demande client.
- Adresse GitHub.
- Zoning.
- Wireframes.
- Maquette Figma publique.
- Schema d'enchainement des ecrans.
- Diagrammes UML.
- Schema de base de donnees.
- Adresse depot GitHub/Docker.
- Scripts de deploiement.
- Scripts CI.
- Configurations Docker.
- Code de l'application.
- Plan de tests.
- README principal.
- Captures de chaque ecran.

## Captures a produire pour Ketsia Shop

- Accueil.
- Catalogue produits.
- Detail ou carte produit.
- Panier.
- Connexion.
- Inscription.
- Profil.
- Commandes.
- Wishlist.
- Paiement.
- Succes paiement.
- Dashboard admin.
- Admin produits.
- Admin commandes.
- Admin utilisateurs.

## Schemas recommandes

- Cas d'utilisation : visiteur, client, administrateur.
- Diagramme de classes : User, Product, Category, Order, OrderItem, Wishlist.
- MCD ou schema BDD.
- Sequence : creation d'une commande.
- Architecture : navigateur, frontend, API, MySQL, Stripe, GitHub Actions, serveur Docker.
